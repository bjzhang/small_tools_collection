# Prompt：语言习惯检查

## 用途
检查技术文章是否符合作者（Bamvor Zhang）的语言习惯，在发布前运行。

## Prompt

请检查以下文章段落是否违反了下列语言习惯规则，逐条列出问题和建议修改：

### 句式禁忌
- **禁用「不是……而是……」**及类似转折句式，换为直接陈述
- **禁用「这不是巧合」**→ 改为「这是个新信号」或类似中性表达
- **禁用「范式转移」「豪赌」「颠覆」「革命性」**等大词，用具体事实替代
- **禁用「正是这种……的体现」**之类的总结性强调句，直接删除或改为事实陈述

### 情绪表达
- 情绪性内容优先使用古文成语，不用现代情绪词
  - 「翻旧账」→「积年之患，一朝显形」
  - 「彻底失效」→「刻舟求剑」（原来有效的经验变得不再适用）
  - 「慢慢失去优势」→「此消彼长」
- 古文成语只用于情绪/感慨段落，技术描述仍用现代汉语

### 数字与数据
- 每个数字必须有来源，不能裸写数字
- 数字改动时，全文同名数字必须同步检查（grep 全文）
- 估算数据需注明「推算」「估算」，不能与实测数据混用

### 英文引文
- 所有英文引文必须附中文译文（blockquote 格式或括号内）
- 译文要求：正常翻译，不得用概括性断语替代原文

### 过渡句
- 过渡句放在章节标题**之后**，不放在标题之前

## 使用方式
将文章段落粘贴到此 Prompt 后，逐段检查输出。
